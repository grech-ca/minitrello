Try it out at: https://minitrello.grech.dev

## Features

* Lists
  - List creation
  - Name editing
  - Drag-and-drop reordering
  - Deletion
* Cards
  - Card creation
  - Name editing
  - Description editing
  - Checlists creation
  - Card modal
  - Drag-and-drop reordering
  - Editable colored labels
  - Description and checklists indicators
  - Checklist progress bar
* Data
  - Local data saved using IndexedDB
  - Ability to reset the app
  
## Technologies
- React
- Typescript
- Redux Toolkit
- Emotion
- React Beautiful DnD
- Framer Motion
